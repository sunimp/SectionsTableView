# SectionsTableView

Simple solution for static or paginating table views.

## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 14.0+

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/sunimp/SectionsTableView.git", .upToNextMajor(from: "1.0.0"))
]
```
